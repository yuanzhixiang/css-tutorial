clip-path Box Model Keywords

这些关键字决定了裁剪形状的参考区域。每个元素都有四层区域：margin（外边距）、border（边框）、padding（内边距）和 content（内容）。 这些关键字告诉浏览器从哪一层开始计算裁剪形状。

margin-box

Content

从最外层的 margin 开始计算裁剪区域
包括了所有的 margin、border、padding 和 content

border-box

Content

从 border 开始计算裁剪区域
不包括 margin，但包括 border、padding 和 content

padding-box

Content

从 padding 开始计算裁剪区域
不包括 margin 和 border，但包括 padding 和 content

content-box

Content

只从 content 区域开始计算裁剪区域
不包括 margin、border 和 padding，只包括 content

Margin (外边距 40px)
Border (边框 20px)
Padding (内边距 30px)
Content (内容区)

理解这些关键字

  • 这些关键字决定了裁剪形状的"参考点"，也就是从哪里开始计算裁剪区域
  • 比如同样是 circle(80%)，在不同的关键字下：
    • margin-box：80% 是相对于包括 margin 在内的整个区域
    • border-box：80% 是相对于从 border 开始的区域（不含 margin）
    • padding-box：80% 是相对于从 padding 开始的区域（不含 margin 和 border）
    • content-box：80% 是相对于只有内容的区域（不含 margin、border 和 padding）
  • 这就像是在告诉浏览器："从这层开始往里裁剪"
